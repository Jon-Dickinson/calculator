## React function based components with hooks and Emotion.

### Simple calculator with basic operations.
### 90 minutes development time.

![](https://github.com/Jon-Dickinson/calculator/blob/master/screenshot.png)

### install packages

#### `yarn`

---

### start app

#### `yarn start`

---

### test app

#### `yarn test`

---

### build app

#### `yarn build`
